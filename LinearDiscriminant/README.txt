Part 1:

To run the classifier to classify the data in 'file.mat'

>> part1 file.mat


Input: file which needs to be classified (testing file)
Output: The labels assigned by all the 5 algorithms.

-------------------------------------------------------------------------

Part 2:

To run the classifier to classify the data in 'file.mat'

>> part2 file.mat <algorithm>

<algorithm> must be 1 for batch perceptron
		    2 for single sample perceptron
		    3 for batch with margin
		    4 for single sample with margin
		    5 for batch relaxation with margin

Input: file which needs to be classified
Output: The labels assigned by the given algorithm number.
